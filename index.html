🖼️ Bildbeschreibung
Was sehe ich?
Auf dem Bild sehe ich …
z.B. „Auf dem Bild sehe ich eine Familie im Park."
📋 Kopiert!
Das Bild zeigt …
z.B. „Das Bild zeigt einen Mann, der am Computer arbeitet."
📋 Kopiert!
Im Vordergrund / Im Hintergrund sieht man …
📋 Kopiert!
Links / Rechts / In der Mitte ist …
📋 Kopiert!
Personen beschreiben
Die Person / Die Personen sieht/sehen … aus.
z.B. „Die Person sieht glücklich aus."
📋 Kopiert!
Ich glaube, die Person ist …
z.B. „Ich glaube, die Person ist eine Ärztin."
📋 Kopiert!
Die Person trägt …
z.B. „Die Person trägt eine Uniform."
📋 Kopiert!
Ort & Situation
Die Situation spielt sich … ab.
z.B. „Die Situation spielt sich in einem Büro ab."
📋 Kopiert!
Ich denke, das ist …
z.B. „Ich denke, das ist ein Supermarkt."
📋 Kopiert!
Es sieht so aus, als ob …
z.B. „Es sieht so aus, als ob die Leute ein Fest feiern."
📋 Kopiert!
Thema benennen
Das Thema des Bildes ist …
z.B. „Das Thema des Bildes ist Gesundheit / Sport / Familie."
📋 Kopiert!
Bei dem Bild geht es um …
z.B. „Bei dem Bild geht es um Ernährung."
📋 Kopiert!
💡 Tipp: Beschreibe das Bild systematisch: erst das Thema nennen, dann die Personen, den Ort und die Details. Benutze Vermutungen wie „Ich denke …" oder „Es sieht so aus, als ob …"
